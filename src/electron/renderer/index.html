How to Use

1. Select Provider

Choose OpenAI or Anthropic as your LLM provider.

2. Enter API Key

Paste your API key. Use "Test Connection" to verify it works.

3. Set Model

Default models: gpt-4o (OpenAI) / claude-sonnet-4-20250514 (Anthropic). You can change to any supported model.

4. Enter Task

Type a natural language instruction in the top bar, e.g. "Open Baidu and search for AI news".

5. Click Start

The AI will control the browser automatically. Watch the logs and screenshots in real-time.

Settings are saved automatically and persist between sessions.
